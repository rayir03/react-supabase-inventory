
# React + Supabase Inventory Starter (Material UI)

Proyecto starter para gestionar productos, lotes y movimientos desde Supabase.

## Cómo usar

1. Descomprime el ZIP.
2. Copia `.env.example` a `.env` y añade tu URL y KEY de Supabase.
3. `npm install`
4. `npm run dev`

## Estructura
- src/
  - supabaseClient.js
  - main.jsx
  - App.jsx
  - components/ (varios componentes)
